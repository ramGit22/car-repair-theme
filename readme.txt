A modern, professional-looking WordPress theme designed specifically for auto repair shops, car services, and mechanics.

== Description ==

Auto Service Pro is a premium-quality WordPress theme tailored specifically for automotive businesses like car repair shops, auto service centers, garages, and mechanics. The theme features a modern design with a focus on usability and lead generation.

This theme includes the following features:

* Mobile-responsive design that looks great on all devices
* Integration with Bookly plugin for online appointment scheduling
* Service showcase section
* Testimonial display
* Contact information and Google map integration
* Schema.org markup for improved SEO
* Customizable color scheme
* Custom page templates
* Print-friendly styles
* Breadcrumb navigation
* Search engine optimized
* Translation-ready

== Installation ==

1. Upload the theme files to the `/wp-content/themes/car-repair-theme` directory, or install the theme through the WordPress "Appearance > Themes" screen and click the "Add New" button.
2. Activate the theme through the "Appearance > Themes" screen in WordPress.
3. Use the Appearance > Customize screen to configure theme settings.

== Creating the Home Page ==

1. Create a new page called "Home" or similar
2. Select the "Home Page" template from the Page Attributes section
3. Publish the page
4. Go to Settings > Reading and set your site to use a static page, selecting the page you just created as the "Home page"

== Bookly Integration ==

This theme works seamlessly with the Bookly plugin for booking appointments:

1. Install and activate the Bookly plugin
2. Configure your services, staff, and other settings in the Bookly admin
3. Create a page with the "Book Now" slug, or edit the existing one
4. Add the Bookly shortcode to the page content: [bookly-form]

== Theme Options ==

This theme includes the following customization options:

1. **Primary Color**: Main theme color used for headings and important elements
2. **Secondary Color**: Accent color used for buttons and highlights
3. **Accent Color**: Support color used for links and minor elements

To modify these settings, go to Appearance > Customize > Theme Colors.

== Shortcodes ==

The theme includes these custom shortcodes:

1. **[contact_form]**: Displays a contact form.
2. **[services_list]**: Displays a list of services.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

The theme is designed to work with the following plugins:
* Bookly (for appointment scheduling)
* Contact Form 7 (for contact forms)
* Yoast SEO (for search engine optimization)
* WP Super Cache (for performance optimization)

= Can I translate this theme? =

Yes, the theme is fully translation-ready and includes the .pot file needed to create translations.

= Is this theme accessible? =

Yes, the theme follows accessibility best practices including keyboard navigation, screen reader text, and proper color contrast.
